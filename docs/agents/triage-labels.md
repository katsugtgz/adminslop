# Triage labels

The triage skill maps its five canonical states to these GitHub labels:

| Canonical role | Label |
| --- | --- |
| `needs-triage` | `needs-triage` |
| `needs-info` | `needs-info` |
| `ready-for-agent` | `ready-for-agent` |
| `ready-for-human` | `ready-for-human` |
| `wontfix` | `wontfix` |

If the GitHub repo does not already have these labels, create them before using the triage workflow.
