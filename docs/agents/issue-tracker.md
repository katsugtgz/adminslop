# Issue tracker

This repo tracks issues in GitHub Issues.

Use the GitHub CLI (`gh`) for issue operations.

## Repository

The GitHub repository is the canonical issue tracker for this project.

## Rules

- Create issues with `gh issue create`.
- Read issues with `gh issue view` and `gh issue list`.
- Apply the triage labels documented in `docs/agents/triage-labels.md`.
- Do not create local markdown issues under `.scratch/` unless the issue tracker configuration is changed.
